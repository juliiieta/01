1.1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

La etiqueta UL sirve para crear listas desordenadas. Es decir que las expresa sin un orden numerico, un ejemplo para usarlo seria una lista de compras.

1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

La etiqueta OL sirve para crear listas ordenadas. Es decir que las expresa con unorden numerico, un ejemplo para usarlo podria ser una lista de pasos a seguir para una receta.

1.3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

La etiqueta DL sirve para crear listas descrptivas. Es decir que expresa una lista de grupo de terminos a los que se le puede dar descripciones, un ejemplo para su uso podria ser una planilla con informacion personal.

1.4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

La etiqueta LI sirve para crear los items de una lista. Es decir que a la hora de usar ol, o ul necesitamos de este.
