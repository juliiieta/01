WEB

En este taller, los estudiantes exploran conceptos de diseño web, lenguajes de marcado (como HTML,JS y CSS) y herramientas de desarrollo para construir páginas web interactivas y visualmente atractivas. Los estudiantes aprenden cómo estructurar contenido, diseñar interfaces amigables y adaptativas, y optimizar la experiencia del usuario en línea. Además, el taller de web puede abarcar aspectos más avanzados, como la introducción al desarrollo web dinámico con lenguajes de programación y la gestión de bases de datos en línea. La finalidad es preparar a los estudiantes para crear y mantener sitios web funcionales y estéticos, lo que es relevante en un mundo donde la presencia en línea es esencial para la comunicación, el comercio y la interacción social.
